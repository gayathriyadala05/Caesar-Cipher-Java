# Caesar Cipher Encryption and Decryption

## 📌 Project Description
This project implements a simple **Caesar Cipher algorithm** using Java.  
The program allows users to input a message and a shift value to **encrypt** and **decrypt** the text.

Caesar Cipher is one of the simplest and most well-known encryption techniques. It works by shifting each letter in the plaintext by a fixed number of positions in the alphabet.

## 🚀 Features
- Encrypts a user input message
- Decrypts the encrypted message
- Supports both **uppercase and lowercase letters**
- Simple and easy to understand Java implementation

## 🛠️ Technologies Used
- Java
- Scanner class for user input

## 📂 How It Works
1. The user enters a message.
2. The user provides a shift value.
3. The program shifts each letter by the given value to encrypt the message.
4. The program then reverses the shift to decrypt the message.

## ▶️ Example
Input:
```
Message: Hello
Shift Value: 3
```

Output:
```
Encrypted Message: Khoor
Decrypted Message: Hello
```

## 📖 Algorithm
1. Read the message from the user.
2. Read the shift value.
3. For encryption:
   - Shift each letter forward by the shift value.
4. For decryption:
   - Shift each letter backward by the shift value.
5. Display encrypted and decrypted messages.
